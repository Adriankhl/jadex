Prebop User Guide

This brief User Guide tells how to use the Prebop preprocessor from an Ant task run from within Eclipse.

Running Ant From Eclipse

  • Use XMLBuddy or the built-in Ant editor to write your Ant project as a file with a .xml extension.
  • Right-click the file name in the Package Explorer or Navigator and select Run Ant...

    This will open a dialog with a number of tabs. Of these, the Target and Refresh tabs are usually relevant.

    Target. The Target tab allows you to specify a target task to run. The default target will be checked and this is usually sufficient.

    Refresh. The Refresh tab allows you to have an Eclipse project or a folder or file within a project refreshed automatically after the Ant task completes. This is useful when the preprocessor will be modifying files in an Eclipse project, so that Eclipse will be using the current versions.

  • The next time you want to run the project with the identical options, you can just select the file from the Run External Tools drop-down list in the toolbar (the icon looks rather distressingly like a child running in front of a car).
