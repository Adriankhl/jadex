License and Versions

The open source software (OSS) files are released under GPL-License. Access to the Pro version and commercial (e.g. redistribution) licensing can be requested from Actoron GmbH.

The features currently described at Jadex Features are available in both the OSS and the Pro version. We will try keeping most functional features publicly available. The Pro version mainly adds non-functional features that we consider relevant for production environments. These features include:

  • Management Extensions (e.g. controlling Jadex platforms remotely from a shell, service pools with automatic load-balancing)
  • Security Features (e.g. SSL message transport)
  • Deployment Support (e.g. automatic load balancing)

Which Distribution to Choose?

Several different packages are available for download. The standard and pro packages include all jars and sources of the main system and the standalone platform. For size reasons, several addons as well as web application deployment files have been excluded from the base package and can be downloaded separately. The addons provide extra features such as 3D visualization and web services support. For a quick-start of using Jadex in a Maven build environment, an example project is provided that can be used for retrieving the required Jadex jars via Maven.
